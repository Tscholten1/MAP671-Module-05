Density of North American Breweries


  • Link to high resolution map

    • Map created by: Tom Scholten, NMP MAP 671

    This project involved a variety of synthesizing data into a usable dataset that visualized point locations of breweries in North America. This involved joining aggregating point locations by using a hexgrids to create a grid network using the MMQGIS plugin in Qgis. The join attributes by location (summary) tool was also utilized to "collect" point locations that were contained within each individual hexagon cell. Considerable time was invested determining the correct settings to create a hexgrid that covered all of North America while also showing an acceptable level of granular detail. Effort was also spent creating the proper visual hierarchy for the maps and ensuring an attractive map product. Coordinate reference system used is North America Equal Area (EPSG: 102008).

    Key lessons learned:

  • Open a variety of geographic data with QGIS and evaluate attribute data within QGIS
  • Utilize the "join" function in Qgis to a spatial and data file (with no spatial geometry)
  • Create a hexgrid using the MMQGIS tool
  • Creating a heatmap and exploring the settings to increase visual appeal of the map
  • Using the join by location (summary) tool
  • Create and export a map layout as a static image file
  • Working with basic HTML elements and CSS style rules to change the web page appearance (e.g. text, lists, page sizing)
  • Working with the Print Layout feature of Qgis to insert a legend, title, text box, arrange map elements
  • Working with the GitHub process to sync files to GitHub remote and create a functional web map page
  • Publishing a static map to a web page
  • Using images as hyperlinks in a dynamic web map
